# lau1yach_Store


### 前端技术：

- 基础的HTML、CSS、JavaScript（基于ES6标准）
- JQuery
- Vue.js 以及基于Vue的框架：Vuetify
- 前端构建工具：WebPack
- 前端安装包工具：NPM
- Vue脚手架：Vue-cli
- Vue路由：vue-router
- ajax框架：axios
- 基于Vue的富文本框架：quill-editor

### 后端技术：

- SpringMVC、Spring和MyBatis
- Spring Boot 
- Spring Cloud
- Redis
- RabbitMQ
- Elasticsearch
- nginx
- FastDFS
- MyCat
- Thymeleaf

### 开发环境
- IDE：IntelliJ IDEA
- JDK：1.8
- 项目构建：maven
- 版本控制工具：git
